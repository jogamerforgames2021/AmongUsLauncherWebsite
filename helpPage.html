How to Download the Launcher

The launcher is a small Windows app that handles everything for you. You download it once, and from that point on it manages game installs, updates, and more.

  1. Head to the homepage
    Go back to the main page and find the big download button.
  2. Click "Download free"
    This grabs the file AmongUsLauncher.exe straight from GitHub.
  3. Save it somewhere you'll find it
    Desktop or Downloads folder works fine.
  4. Run it
    Double-click the file and the launcher will open up.
Windows SmartScreen warning Windows might warn you because it doesn't recognise the app yet. Just click "More info" then "Run anyway." The launcher is open source and safe to run.
Download the Game Directly

Prefer to skip the launcher? You can grab any version of the game as a zip file directly from the homepage.

  1. Find the version picker
    On the homepage, below the main download button, there's a small panel that says "Just want the game files?"
  2. Pick a version
    Click the dropdown to see all available versions. The latest one is marked at the top.
  3. Download and unzip
    Click "Download game zip", extract the folder wherever you like, and run Among Us.exe inside it.
Heads up The launcher is easier to use long-term since it handles updates for you. Direct downloads are great if you just need a specific version quickly.

Getting Started

Once you've downloaded the launcher, setup takes less than two minutes. Here's what to expect the first time you open it.

  1. Open the launcher
    Double-click AmongUsLauncher.exe. A window will open with a clean interface.
  2. Internet check
    The launcher checks your connection and looks for any updates to itself. This takes a second or two.
  3. You're in
    You'll see the main screen with a big install button, your version info, and a sidebar for settings and tools.
Worth knowing Your settings and game path are saved between sessions. You only have to set things up once.
Installing the Game

The launcher downloads and sets up Among Us for you. Here's how the first install goes.

  1. Click "Install Game"
    It's the big button in the middle of the launcher window.
  2. Choose an install folder
    Pick somewhere sensible like C:\Games\Among Us. You can change this later in settings.
  3. Wait for the download
    You'll see a progress bar with a real-time speed indicator. Shouldn't take long on a decent connection.
  4. Automatic setup
    The launcher extracts everything into place on its own. Nothing else to do.
  5. Launch button appears
    Once it's done, the button flips to "Launch Game." You're good to play.
Auto Updates

Keeping the game up to date is completely automatic. You don't have to check anything or visit the website again.

  • Every time you open the launcher, it checks for a newer version in the background.
  • If something new is available, the main button turns into "Update Available."
  • Click it once and it downloads and installs the update by itself.
  • Every download includes a checksum check so corrupted files are caught automatically.
Version Management

Need to play on an older version? Maybe your friends are still on a build from a few months back. The launcher makes it easy to switch.

  1. Click "Install Specific"
    You'll find this next to the main action button in the Game tab.
  2. Browse the list
    A window opens showing every available version, newest first.
  3. Pick one and install
    Click the version you want and hit "Install Selected." That's it.
This replaces your current install Switching versions overwrites what's currently installed. If you've got any save data you care about, back it up before switching.
Discord Rich Presence

When you're in-game, your Discord status can show that you're playing Among Us along with which version you're on. Friends can see it at a glance.

  1. Turn it on in Settings
    Open the sidebar menu, go to Settings, and enable "Discord Rich Presence."
  2. Make sure Discord is running
    Discord needs to be open before you launch the game or it won't pick up the status.
  3. It just works from there
    Your status will automatically show "Playing Among Us" with the version info.
What shows up Your status shows "Playing Among Us Shadow Slime" with your current version and whether you're in the launcher or actually in-game.
Installing AUnlocker

AUnlocker is an optional mod that unlocks all cosmetics and lets you use characters like Russian letters in your player name. It only touches visual stuff and doesn't affect gameplay or multiplayer.

  1. Open the launcher
    Go to the Tools or Mods section from the sidebar.
  2. Hit "Install AUnlocker"
    It downloads and installs automatically. No files to move around yourself.
  3. Launch the game normally
    Everything will be unlocked the next time you start.
It's safe AUnlocker is cosmetics only. It doesn't touch personal data, doesn't contain malware, and is verified when installed through the launcher.
Desktop Shortcuts

You can create a desktop shortcut to launch a specific version of the game directly, without opening the full launcher UI first.

  1. Select the version you want a shortcut for
  2. Click "Create Desktop Shortcut"
    The button is in the version management view.
  3. Done
    A shortcut appears on your desktop. Double-click it to go straight in.

Troubleshooting

Most issues have a pretty simple fix. Try these first before heading to Discord.

  • Launcher won't open — Right-click the .exe and choose "Run as administrator."
  • Game won't launch — Use the launcher to reinstall the game. This usually fixes any missing or corrupted files.
  • Antivirus deleting files — Add the launcher folder and game folder to your antivirus exclusions, then reinstall the game.
  • Stuck on infinite loading screen — Check you have all the required redistributables installed (see the section below).
  • Discord status not showing — Close Discord and reopen it, then relaunch the game.
  • Download keeps failing — Try pausing your antivirus during the download, then add the folder to exclusions afterwards.
Still stuck? Jump into the Discord and describe what's happening. Include your Windows version and what error you're seeing if there is one.
Required Software

If Among Us crashes on startup, won't open at all, or gets stuck loading forever, the most common cause is missing system components. These are small packages that the game needs in order to run.

Think of them like drivers. Even on a clean Windows install, some of these might not be there yet. Installing them takes a few minutes total and often fixes the problem immediately.

Install both x86 and x64 where available Even if your system is 64-bit, some games need the 32-bit versions too. Install both to be safe.

A few extra things to check:

  • Set a default browser — Some launcher features use it. If you don't have one set, go to Windows Settings and pick one.
  • Run as administrator — Right-click Among Us.exe and choose "Run as administrator" if it keeps failing to start.
  • Disable antivirus during extraction — Real-time protection can silently delete files mid-extraction. Pause it, extract, then add the folder to exclusions.
